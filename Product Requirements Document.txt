Создай полностью готового к продакшену Telegram-бота для торговли внутриигровыми автомобильными номерами игры Car Parking Multiplayer 2.

Бот представляет собой маркетплейс цифровых игровых номеров.

Все номера являются исключительно внутриигровыми цифровыми активами и не имеют никакого отношения к реальным государственным регистрационным знакам Российской Федерации, Республики Казахстан и других стран.

Код должен быть готов к запуску в продакшене без необходимости полной переработки архитектуры.

Используй чистую архитектуру, масштабируемую структуру проекта и современные практики разработки.

# Концепция платформы

Платформа представляет собой биржу игровых номеров для Car Parking Multiplayer 2.

Пользователи могут

 покупать новые номера у Государства;
 продавать номера другим игрокам;
 участвовать в аукционах;
 пополнять баланс через Telegram Stars;
 искать редкие номера;
 собирать коллекции.

Каждый номер существует только в одном экземпляре.

Например

А777АА77

777AAA01

Если номер уже существует, повторная эмиссия невозможна навсегда.

# Защита от Unicode spoofing

Для Казахстана разрешён только ASCII Latin.

Запретить

 Unicode lookalike символы;

# Нормализация ввода

# Эмиссия номеров

База данных изначально пустая.

показать предложение купить номер у Государства за ⭐1.

После оплаты

создать новый номер в системе.

Цена государственной эмиссии должна изменяться через админ-панель.

# Продажа номеров

Пользователь может выставить номер на продажу.

Ограничения

 только целые числа;
 минимум ⭐1;
 максимум ⭐99999.

После продажи

90% получает продавец.

10% получает Государство.

Комиссия должна изменяться через настройки администратора.

# Баланс

Необходимо реализовать

available_balance

frozen_balance

В базе хранить целые числа.

Отображать

⭐150

# Telegram Stars

Поддерживать официальную систему Telegram Stars.

После успешной оплаты

увеличивать доступный баланс пользователя.

# Аукционы

Максимальная длительность

24 часа.

Минимальный шаг

⭐5.

Параметры должны изменяться через настройки.

После ставки

деньги замораживаются.

Если ставку перебили

предыдущая ставка автоматически размораживается.

Отмена аукциона разрешена только если отсутствуют ставки.

Отмена ставок запрещена.

# Anti-sniping

Если ставка сделана менее чем за 5 минут до окончания

аукцион продлевается на 5 минут.

Время продления должно быть настраиваемым.

# Резервирование

Во время выставления счёта номер резервируется на 5 минут.

Использовать

reserved_by

reserved_until

Просроченные резервы автоматически удаляются.
